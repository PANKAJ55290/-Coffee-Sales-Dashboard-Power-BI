☕ Coffee Sales Dashboard | Power BI
📌 Project Overview

The Coffee Sales Dashboard is an interactive Business Intelligence dashboard developed using Microsoft Power BI. The objective of this project is to transform raw coffee shop sales data into meaningful insights that help business stakeholders monitor sales performance, identify customer purchasing behavior, and make data-driven decisions.

The dashboard provides a comprehensive overview of key business metrics such as Total Sales, Total Transactions, Average Sales, Monthly Sales Trends, Sales by Time of Day, and Product-wise Sales Performance.

🎯 Project Objective

The primary objective of this project is to:

Monitor overall sales performance.
Analyze monthly sales trends.
Identify the highest-selling coffee products.
Understand customer purchasing behavior throughout the day.
Build an interactive dashboard for business decision-making.
Demonstrate practical Business Intelligence skills using Power BI.

The dataset was imported into Power BI for analysis and visualization.

🔄 Data Preparation

Before creating the dashboard, the dataset was prepared using Power Query.

The following data-cleaning steps were performed:

Imported Excel dataset into Power BI.
Checked and corrected data types.
Removed duplicate records.
Handled missing values.
Renamed columns for consistency.
Prepared data for reporting and visualization.
